
# 商务合作

目前SDK分为商业版和开放版。

### 开放版

github中的demo使用的为开放版sdk,开放版SDK仅供研究或个人使用，开放版有调用次数限制，请不要在上线的APP中使用开放版SDK.

### 商业版

正式上线的APP需要授权，授权后集成商业版SDK, 集成商业版SDK的APP将获取稳定的服务以及持续的技术支持。
